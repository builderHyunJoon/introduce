삼성 RCS를 활용하여 회원가입과 로그인 없이 챗봇으로 미리 주문할 수 있는 스몰오더 서비스 프로젝트에 투입되어 운영 및 고도화 개발을 진행하였습니다. 서비스 운영과 담당자 및 협력 관계 개발자들과의 커뮤니케이션, Jenkins와 K8S를 활용한 CI/CD 파이프라인 구축 및 서비스 무중단 배포를 직접 경험하고 익혔습니다. 프로젝트를 통하여 경험한 기술 스택과 사용한 툴은 아래와 같습니다.

  • Programming Language: Java8, JS
  • Framework: Spring Boot
  • Template Engine: Thymeleaf
  • DBMS: PostgreSQL
  • ORM: MyBatis
  • Build Tool: Maven
  • CI/CD Tool: Jenkins
  • Container Orchestration: Kubernetes, OpenShift
  • Server OS: Red Hat Enterprise Linux
  • Version Control System: GitLab
  • IDE: Spring Tool Suite 4
  • Code Editor: VSCode
  • API Testing Tool: Postman
  • Database Access Tool: pgAdmin 4
